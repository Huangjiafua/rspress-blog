# MySQL 学习导航

这里整理 MySQL 学习的核心内容，按「先基础、再函数」的顺序推进。

## 学习路径

1. 先掌握 SQL 基础语法与数据类型、约束概念。
2. 再学习常见函数，提升查询与数据处理效率。
3. 结合实际表结构做查询练习，形成可复用 SQL 模板。

## 目录

- [基础语法](./basic-grammar)
- [常用函数](./function)

## 学习建议

- 每个知识点都至少手写并执行 3 条 SQL。
- 涉及更新和删除语句时，先写 `SELECT` 验证条件再执行。
- 事务练习优先用 `START TRANSACTION` + `ROLLBACK` 做安全演练。
